Function Search(head, key):
    current = head
    
    While current is not NULL:
        If current.data == key:
            Return current
        else if current.data != key:
            current = current.next    
    Return NULL

2. Write an algorithm to sorting a singly linked list.
Function BubbleSort(head):
    If head == NULL:
        Return head

    swapped = true

    While swapped == true:
        swapped = false
        current = head

        While current->next != NULL:
            If current.data > current->next.data:
                // Swap the data of the two nodes
                temp = current.data
                current.data = current->next.data
                current->next.data = temp
                swapped = true
            Else If:
            current = current->next
    Return head

3. Why reverse algorithm is not needed in doubly linked list. Justify your answer comparing with singly linked list.

In a **doubly linked list**, each node has pointers to both its **next** and **previous** nodes. This allows easy traversal in both directions (forward and backward). Therefore, you can move backward through the list directly using the `previous` pointer without needing an explicit reverse algorithm.

In contrast, in a **singly linked list**, each node only has a pointer to the **next** node, meaning traversal can only happen in one direction (forward). To reverse the list, you must modify the pointers to allow backward traversal, which is why a reverse algorithm is necessary.

### Comparison:
- **Doubly Linked List**: Direct backward traversal using `previous` pointer. No reverse algorithm needed.
- **Singly Linked List**: Only forward traversal possible. Reverse algorithm required to enable backward traversal.

4.. Write an algorithm to delete the entire singly linked list or doubly linked list.
Function DeleteList(head):
    current = head
    
    While current != NULL:
        nextNode = current->next  // Store the next node
        Free(current)  // Delete the current node
        current = nextNode  // Move to the next node
    End While

    head = NULL  // Set head to NULL after deleting all nodes
End Function
